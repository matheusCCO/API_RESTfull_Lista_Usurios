#  API RESTful - Gerenciamento de Usuários

## Sobre o Projeto

Este projeto é uma API REST desenvolvida em **Java** utilizando **Spring Boot**, com o objetivo de implementar um CRUD completo para gerenciamento de usuários.

A aplicação foi construída seguindo boas práticas de desenvolvimento backend, utilizando arquitetura em camadas (**Controller, Service e Repository**), separação de responsabilidades e organização clara do código.

> Projeto desenvolvido com foco em prática profissional de backend e composição de portfólio.

---

## Tecnologias Utilizadas

- Java 17+
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven
- Banco de dados (H2 ou similar)

---

## Conceitos Aplicados

Este projeto demonstra conhecimento em:

-  Construção de APIs REST
-  Padrão de arquitetura em camadas
-  Injeção de dependências
-  Manipulação de requisições HTTP (GET, POST, PUT, DELETE)
-  Serialização e desserialização JSON
-  Integração com banco de dados via JPA
-  Organização de projeto Maven

---

## Funcionalidades

A API permite:

-  Criar um novo usuário  
-  Listar todos os usuários  
-  Buscar usuário por ID  
-  Atualizar informações de um usuário  
-  Remover usuário da base de dados  

Todas as respostas são retornadas em formato JSON.
